## Risk Register (carried in REQ tickets)

| Risk | Mitigation | Owner |
|---|---|---|
| `endoflife.date` API goes down or rate-limits us | Circuit-breaker + 24h cache (REQ-1-AC5 + REQ-3-AC4) | Backend |
| LLM hallucinates a "high-impact" signal causing alert-fatigue | Confidence threshold 0.5 (reused from UC-ICM-002), dismiss-feedback-loop tracked in metrics | Product |
| Cron drift across timezones (DST issues) | Pin `0 6 * * *` to Europe/Berlin explicitly, document in code | Backend |
| Signal-storm on first scan (1000+ historic deadlines / EOLs) | First-run flag: only emit signals with `timeToImpactDays ≤ 365` initially; full history is opt-in | Backend |
| Compliance-deadline data goes stale (laws change) | Manual quarterly review process documented in `compliance-deadlines.json` header | Legal / Product |
