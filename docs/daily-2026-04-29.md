## Lessons

- **Pre-Flight ist Gold.** THE-83 hatte WSJF-Aufwand 1d, tatsächlicher Aufwand 30 Min. Ohne Pre-Flight 4-6h auf gelöste Probleme geworfen. Linear ↔ Code-Realität war signifikant gedriftet.
- **"Toast grün" beweist nichts.** MiroFish-Sync hat einen Tag lang freudig "synced" gemeldet, während stumme Validation-Errors 11 Personas verworfen. Surface für Failures muss Teil von Bulk-APIs sein.
- **304-Cache + Auto-Sync-Recursion = perfide.** Server cached leere Liste, Client sieht Cache, triggert Sync, Server skipt Duplikate, Client cached weiter leer. Reentrancy-Guard + `Cache-Control: no-store` brechen das.
- **Heads-up bei AI-extracted Stakeholdern:** das `role`-Feld ist eine ganze Beschreibung, kein Titel. Defensive Clamps an Persistence-Boundaries sind Pflicht.
